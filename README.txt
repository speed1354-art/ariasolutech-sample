아리아솔루텍 홈페이지 시안

실행 방법
1. 압축을 풉니다.
2. ariasolutech-site 폴더를 VS Code로 엽니다.
3. index.html을 더블클릭하거나 Live Server로 실행합니다.

구성
- index.html : 메인
- company.html : 회사소개
- business.html : 사업분야
- technology.html : 기술역량
- project.html : 프로젝트
- contact.html : 문의
- css/style.css : 전체 디자인
- js/main.js : 모바일 메뉴 및 스크롤 애니메이션
- robots.txt / sitemap.xml / site.webmanifest : 기본 SEO

디자인 방향
- 어두운 배경 전면 배제
- 화이트, 실버, 블루그레이
- 반투명 글래스 패널
- 넓은 여백과 부드러운 카드 전환
- 모바일 반응형 적용

SEO
- 페이지별 title / description
- canonical
- Open Graph / Twitter Card
- Organization JSON-LD
- robots.txt / sitemap.xml

중요
현재 도메인은 임시값입니다:
https://www.ariasolutech.co.kr

실제 도메인이 정해지면 VS Code 전체 검색(Ctrl+Shift+F)으로 위 주소를 찾아 실제 도메인으로 일괄 교체하세요.

images/og-image.jpg는 직접 추가해야 합니다.
권장 크기: 1200 × 630px

연락처, 이메일, 연혁, 인증, 프로젝트 내용은 실제 업체 확인 후 교체해야 합니다.
문의 폼은 디자인만 적용되어 있으며 실제 전송 기능은 없습니다.
